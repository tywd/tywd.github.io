# 欢迎来到我的博客

这是我的第一篇博客文章，主要记录一下博客的搭建过程。

## 搭建过程

1. 初始化项目目录
2. 安装 VitePress
3. 配置基本设置
4. 创建首页和基本页面结构

## 遇到的问题

在搭建过程中遇到了一些问题：

1. 依赖安装速度较慢
2. 配置文件的路径需要注意
3. 部署到 GitHub Pages 需要特殊配置

## 解决方案

针对上述问题，我采用了以下解决方案：

1. 使用国内镜像源加速依赖安装
2. 仔细阅读官方文档确认配置项
3. 配置正确的 base 路径用于部署

希望这个博客能帮助到更多的人！
